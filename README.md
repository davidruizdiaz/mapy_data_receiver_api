# VCounter API

## Descripción

API para obtener datos del contador de vehículos.

### Dependencias

| Dependencia | Versión |
| ----------- | ------- |
| nodejs      | 16.20.2 |
| fastify     | ^4.29.0 |
| knex        | ^3.1.0  |
| oracledb    | ^6.8.0  |
| jest        | ^29.7.0 |

### Construcción (build)

Ejecute `node build_scripts/builder` para generar una carpera dist/ para distribución.
